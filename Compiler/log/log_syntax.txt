Terminals symbols: 
	- 'begin', 'end', 'msec', 'sec', ';', ':', '(', ')', 'move' //write the other actions

Nonterminals symbols:
	- Program (start symbol), Body, Timestamp, Time-Value, Unit, Identifier, Action, Parameter-Value
Production rules :

Program ::= begin Body end
Body ::= Line+    // = Line [Line]*
Line ::= Identifier Action '('Parameter')' ';'
Identifier ::= Letter [Letter|[0-9]|_]*
Parameter ::= Parameter-Value [','Parameter-Value]*
Parameter-Value ::= Letter Parameter-Value | [0-9] Parameter-Value| E(empty)
Action ::= move | 
Timestamp ::= Time-Value Unit
Time-Value ::= [1-9][0-9]*
Unit ::= sec | msec
Letter ::= [a-z][A-Z]




example:

2 s : robot_test moved (2,3);
